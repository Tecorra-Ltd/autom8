## Description

[Autom8](https://github.com/Tecorra-Ltd/autom8) The world's first automation platform designed for people who want to live more and work less. Let's focus on what's important in life.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
